NINA – AI-Controlled Voice Assistant
NINA is an AI-powered voice assistant that enables system-level task execution on a laptop using voice commands.
The assistant processes spoken input, understands intent using AI, and performs real-time automation tasks.


Features
Voice-controlled system automation
Email composition and reminder setup
Browser search and location-based queries
Weather and traffic information
Screenshot capture
Media control (YouTube playback)
Question solving and code generation
Modular command execution architecture


Tech Stack
Python – core backend and automation logic
Gemini API – intent understanding and AI responses
SpeechRecognition – voice input processing
WebSockets – real-time communication between backend and GUI
JavaScript & CSS – frontend GUI

How It Works
Voice input is captured using the microphone
Speech is converted to text
Intent is identified using Gemini API
Commands are routed to relevant automation modules
Actions are executed on the system in real time
